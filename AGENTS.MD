# AGENTS.MD

Ini adalah file dokumentasi untuk agen-agen yang digunakan dalam proyek ini.

## Agen 1: LangChain Agent
- Menggunakan framework LangChain.
- Dapat mengeksekusi tugas berdasarkan instruksi pengguna.

## Agen 2: LM Studio Provider Agent
- Menggunakan provider LM Studio.
- Menyediakan akses ke model-model besar.

## Agen 3: File System Agent
- Mengakses dan mengelola file sistem.
- Memungkinkan operasi seperti membaca, menulis, menghapus, dan melihat isi folder.

## Agen 4: AI Assistant Agent
- Memberikan bantuan dan penjelasan.
- Dapat menjawab pertanyaan umum dan membantu pengguna.

## Agen 5: Tool Executor Agent
- Menggunakan tool untuk mengeksekusi tugas tertentu.
- Dapat memanggil tool seperti `read_file`, `write_file`, `update_file`, dll.

## Agen 6: Contextual Agent
- Menyediakan konteks berdasarkan riwayat percakapan.
- Dapat mengingat dan menggunakan informasi dari percakapan sebelumnya.

## Agen 7: Rule-Based Agent
- Menggunakan aturan tertentu untuk menentukan tindakan.
- Dapat mengambil keputusan berdasarkan kondisi tertentu.

## Agen 8: Prompt Generator Agent
- Menghasilkan prompt yang sesuai dengan kebutuhan.
- Dapat menghasilkan prompt untuk model besar.

## Agen 9: Error Handler Agent
- Menangani kesalahan dan memberikan respons yang tepat.
- Dapat mengidentifikasi dan menangani error yang mungkin terjadi.

## Agen 10: Logging Agent
- Menyimpan log aktivitas.
- Dapat mencatat aktivitas pengguna dan tindakan yang dilakukan.


---

> *Catatan: Agen-agen ini bisa diaktifkan atau dinonaktifkan sesuai kebutuhan.*
